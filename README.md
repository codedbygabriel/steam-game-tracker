# Steam Progress Tracker

> Simple Website To Track Your Games Status!

## Usage

Whenever you feel lost in your **steam** library use this webite! It'll let you organize which ones you have completed, abandoned, in progress or unregistered.

> Acceptable arguments:

- https://steamcommunity.com/id/${user}
- https://steamcommunity.com/id/${id}
- ${id}
- ${user}
